CSS3 Box Model Demo

This is the main content area.

On large screens the navigation menu is shown in a panel on the left.

On smaller screens (mobile) the navigation is hidden, but the top navigation bar contains a button that reveals the navigation in a hidden panel.

Both the navigation menu and this text are contained in vertical scrollboxes. This means that if they don't fit on the screen, they will be scrollable--independent of each other

Dialog

This modal dialog is adapted from the standard Bootstrap modal.

The main changes are:

  1. The body area will size in height to accommodate its content--as long as it fits on the screen, beyond which the body area will become scrollable
  2. On smaller screens, the dialog will take up the full width and height of the screen to let no space go wasted.
